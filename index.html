awesome-colorterm

A curated list of awesomescripts, programs, and other functions for a colorful terminal environment.

Info

You'll probably want to set your terminal type to xterm-256color or screen-256color.

Contents

  • Colorful wrapper scripts & alternatives to common terminal commands
  • Parameters for common commands to enable colored output
  • Color definitions for GRC
  • Color definitions for various terminal applications (PuTTY, xterm, etc)
  • Color definitions for various shells
  • Other Resources

Colorful wrapper scripts & alternatives

  • cdu - (for Color du) is a perl script which call du and display a pretty histogram with optional colors which allow to imediatly see the directories which take disk space.
  • grc - yet another colouriser (written in python) for beautifying your logfiles or output of commands.
  • dfc - displays file system space usage using graphs and colors. Based on df. able to use colors, draw graphs and export to CSV, HTML.
  • nojhan/liquidprompt - for bash and zsh
  • colormake - a perl wrapper for make that colorizes the output
  • colormake - a python wrapper for make that colorizes the output
  • trapd00r/File-LsColor - Colorize input filenames just like ls(1) does
  • colorex - display files or sdtin with pretty colors for matched patterns.
  • rainbow - Colorize commands output or STDIN using patterns. (fork of colorex)
  • tput - move the cursor around the screen, set colors. examples
  • tcolors
  • colortest - utilities to test color capabilities of terminal
  • multitail - allows you to monitor logfiles and command output in multiple windows in a terminal, colorize, filter and merge

Parameters

  • less - use -r to include color (and other escapes)
  • grep - use --color=always or --color=auto
  • ls - use --color=always or --color=auto
